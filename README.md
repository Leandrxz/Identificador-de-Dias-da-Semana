**Dia da Semana em Java.**

**Descrição:**
Este projeto em Java implementa um programa simples que solicita ao usuário a entrada de um número de 1 a 7 e exibe o dia correspondente da semana.

A lógica do programa é baseada na estrutura de controle switch, associando cada número a um dia específico.

Este exercício tem como objetivo praticar conceitos básicos da linguagem Java, como:
Entrada de dados com Scanner
Estruturas condicionais (switch-case)
Interação com o usuário via terminal

Funcionamento:
O programa solicita que o usuário digite um número de 1 a 7.

Com base no número informado, o sistema retorna o dia da semana correspondente:
>1 – Domingo
>2 – Segunda-feira
3 – Terça-feira
4 – Quarta-feira
5 – Quinta-feira
6 – Sexta-feira
7 – Sábado
Caso o número esteja fora do intervalo válido, é exibida a mensagem "Opção inválida".

**Tecnologias Utilizadas:**
Java
Scanner (java.util)
